Cookies can be manipulated via JavaScript - the cookies listed above are extrated using JavaScript. However, if a cookie has the HTTPOnly flag, it won't be visible to JavaScript, as you can try by visiting secret. The cookie contains 'message=shhh', but it isn't on the list above. However, you can see it with the web tools (right-click, inspect, and view the headers under the network tab).

One other important flag is the session flag - cookies marked as session cookies are discarded once the browser closes.
